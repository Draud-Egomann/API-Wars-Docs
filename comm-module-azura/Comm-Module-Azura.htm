$ curl -XPOST http://192.168.100.13:2030/put_message --data '{"sending_station": "Shangris Station", "base64data": "dGVzdAo="}'
{ "kind": "success" }

$ curl http://192.168.100.13:2030/messages_for_other_stations
{ "kind": "success", "received_messages": [{"dest": "Shangris Station", "base64data": "dGVzdAo="}] }
